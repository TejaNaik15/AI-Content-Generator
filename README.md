# 🤖 AI Content Generator

<div align="center">

![AI Content Generator](https://img.shields.io/badge/AI%20Content%20Generator-v1.0.0-blue?style=for-the-badge)
![License](https://img.shields.io/badge/License-MIT-green?style=for-the-badge)

A modern, full-stack AI-powered blog post generator with user authentication, credit management, and responsive design. Built with React and Node.js, featuring mobile-optimized components and cross-device compatibility.

## ✨ Features

- 🤖 **AI Content Generation**: Generate high-quality blog posts with customizable tone and category
- 🔐 **User Authentication**: Secure registration, login, and session management
- 💳 **Credit System**: Track and manage content generation credits
- 📱 **Responsive Design**: Optimized for desktop, tablet, and mobile devices
- 📚 **Content History**: View and manage previously generated content
- 💰 **Payment Integration**: Stripe integration for premium plans
- 📲 **Mobile-First**: Special mobile components for iPhone and Android compatibility
- ⚡ **Real-time Updates**: Dynamic credit tracking and user profile management

## 🚀 Getting Started

### Prerequisites

<div align="center">

![Node.js](https://img.shields.io/badge/Node.js-v16+-339933?style=for-the-badge&logo=node.js&logoColor=white)
![MongoDB](https://img.shields.io/badge/MongoDB-v4.4+-47A248?style=for-the-badge&logo=mongodb&logoColor=white)
![Git](https://img.shields.io/badge/Git-Latest-F05032?style=for-the-badge&logo=git&logoColor=white)

</div>

- Node.js (v16 or higher)
- MongoDB (local or cloud instance)
- Git

### 📥 Installation

1. **Clone the repository**
```bash
git clone https://github.com/yourusername/ai-content-generator.git
cd ai-content-generator
```

2. **Backend Setup**
```bash
cd ai/backend
npm install
```

3. **Frontend Setup**
```bash
cd ai/frontend
npm install
```

4. **Environment Variables**

Create `.env` files in both frontend and backend directories:

**Backend (.env):**
```env
MONGODB_URI=your_mongodb_connection_string
JWT_SECRET=your_jwt_secret_key
OPENAI_API_KEY=your_openai_api_key
STRIPE_SECRET_KEY=your_stripe_secret_key
PORT=5000
NODE_ENV=development
```

**Frontend (.env):**
```env
VITE_API_URL=http://localhost:5000
VITE_STRIPE_PUBLISHABLE_KEY=your_stripe_publishable_key
VITE_APP_NAME=AI Content Generator
```

### 🏃‍♂️ Running the Application

1. **Start Backend Server**
```bash
cd ai/backend
npm start
# or for development
npm run dev
```

2. **Start Frontend Development Server**
```bash
cd ai/frontend
npm run dev
```

3. **Access the Application**
- 🌐 Frontend: `http://localhost:3000`
- 🔌 Backend API: `http://localhost:5000`

## 📊 API Endpoints

### 🔐 Authentication
- `POST /api/v1/users/register` - User registration
- `POST /api/v1/users/login` - User login
- `POST /api/v1/users/logout` - User logout
- `GET /api/v1/users/auth/check` - Check authentication status
- `GET /api/v1/users/profile` - Get user profile

### 🤖 Content Generation
- `POST /api/v1/ai/generate` - Generate AI content
- `GET /api/v1/users/history` - Get content history

### 💳 Payments
- `POST /api/v1/stripe/create-checkout-session` - Create Stripe checkout
- `POST /api/v1/stripe/webhook` - Handle Stripe webhooks

## 🎨 Key Features Implementation

### 💳 Credit System
- localStorage-based credit tracking

- Visual credit indicators and warnings

### 📱 Mobile Compatibility
- Automatic mobile device detection

### 🔐 Authentication Flow
- JWT-based authentication

## 🔧 Customization

### Adding New Content Categories
1. Update the category options in `ContentGeneration.jsx`
2. Modify the content generation logic
3. Add category-specific styling if needed

### Modifying Credit Limits
1. Update the default credit values in components
2. Modify backend validation if implemented
3. Update pricing plans accordingly

### Styling Customization
1. Modify the gradient colors in components
2. Update the MobileSafeBackground component
3. Adjust responsive breakpoints and clamp() values
